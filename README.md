1. ##### Работа с git'ом

###### Базовые команды:

git init - инициализация git-репозитория в текущей папке

git clone <ссылка> - клонирование и инициализация git-репозитория в новую папку

git clone https://github.com/Esyenin/DigitalDiaryApp.git - клонирование нашего git-репозитория 

git status - текущее состояние репозитория 

git add <файл> - добавление измененного файла к новому commit'у (если написать вместо <файл> обычную точку, то добавятся все изменения)

git commit -m <комментарий> - создание нового коммита, куда войдут все изменения (комментарий обязателен, осмысленен)

git push - отправление коммита в систему контроля версий

git pull - получения последних изменений (commit'а) в текущей ветке

git switch - переключение между ветками
